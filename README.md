O projeto foi configurado com o XR Plugin Management ativo para PC/Desktop conforme as diretrizes do edital, a navegação inicial e a avaliação do ambiente podem ser feitas diretamente pelo Unity Editor através da aba "Scene".

Comandos de Navegação no Unity Editor (Aba Scene):
1. Olhar ao redor (Rotacionar a Visão): Segure o botão DIREITO do mouse e mexa o mouse para os lados, para cima ou para baixo.
2. Zoom (Aproximar ou Afastar): Use a rodinha do mouse (Scroll) para se aproximar ou se afastar dos objetos.
3. Mover a câmera (Pan): Segure a rodinha do mouse (clique nela como se fosse um botão) ou segure a tecla ALT + botão ESQUERDO do mouse e arraste para os lados para navegar pelo plano.
